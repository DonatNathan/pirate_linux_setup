# Pirate Linux Setup

Here are some simple scripts that setup my linux config.

## Install

### How to install my ubuntu setup
```
git clone git@github.com:DonatNathan/pirate_linux_setup.git
cd pirate_linux_setup
./setup_ubuntu.sh
cd ..
```

## Uninstall and clear

```
./clear_setup.sh
```
